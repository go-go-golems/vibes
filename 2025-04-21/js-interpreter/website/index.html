JavaScript Testing with Goja Interpreter

Exploring JavaScript language features in a minimalistic VM environment

Project Overview

This project demonstrates the capabilities of the Goja JavaScript interpreter, a Go-based JavaScript virtual machine. We've created 21 JavaScript programs that test various language features and run them through our custom Goja interpreter implementation.

Test Success Rate: 100% (21/21 programs)

Each program focuses on a specific JavaScript feature or concept, from basic arithmetic to advanced features like generators and promises. The tests verify that the Goja interpreter correctly implements JavaScript language features according to the specification.

Goja Interpreter Implementation

Our custom Goja interpreter is implemented in Go and provides a minimalistic JavaScript VM environment. It includes:

  • Console module for output capture
  • Detailed error handling and reporting
  • Performance metrics tracking
  • Comprehensive test result generation

The interpreter successfully ran all 21 JavaScript programs with a 100% success rate, demonstrating the robustness of the Goja JavaScript engine.
