Form Filling Co-Pilot

This is a static demonstration of the Form Filling Co-Pilot application.

About This Project

This application demonstrates a chat-based form filling assistant built with:

  • Frontend: Next.js with assistant-ui library
  • Backend: Golang with Gin framework
  • Features: Chat interface, form filling assistance, form submission

The assistant helps users fill out forms by guiding them through the process in a conversational manner.

Form Interface

The form interface allows users to enter their information directly.

Chat Interface

The chat interface allows users to interact with an AI assistant to fill out the form.

Filled Form

Example of a form filled out with user information.

Form Submission

The form can be submitted to the backend for processing.

Architecture

Frontend

  • Next.js framework
  • assistant-ui library for chat interface
  • Tailwind CSS for styling
  • React Hook Form for form handling

Backend

  • Golang programming language
  • Gin web framework
  • RESTful API endpoints
  • Mock AI response generation
