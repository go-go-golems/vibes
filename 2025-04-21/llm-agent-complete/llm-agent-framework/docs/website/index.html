Build Intelligent Agents with Go

GoAgent is a flexible, extensible framework for building LLM-powered agents in Go. Create agents that can research, plan, and solve complex problems.

Get Started View Examples

Key Features

Multiple Agent Patterns

Implement ReAct, Plan-and-Execute, and other agent patterns to solve different types of problems.

Vector Memory

Store and retrieve information using vector embeddings for context-aware agents.

Tool Integration

Easily extend your agents with custom tools to interact with external systems.

Execution Tracing

Debug and analyze agent behavior with detailed execution traces.

Getting Started

# Install the framework
go get github.com/goagent/framework
package main

import (
    "context"
    "fmt"
    "github.com/goagent/framework/goagent/agent"
    "github.com/goagent/framework/goagent/llm"
    "github.com/goagent/framework/goagent/tools"
)

func main() {
    // Create a mock LLM
    mockLLM := llm.NewMockLLM()
    
    // Create a ReAct agent
    reactAgent := agent.NewReActAgent(mockLLM, 10)
    
    // Add a tool
    webSearch := tools.NewWebSearchTool()
    reactAgent.AddTool(webSearch)
    
    // Run the agent
    ctx := context.Background()
    result, _ := reactAgent.Run(ctx, "Search for information about Go programming")
    
    fmt.Println(result)
}
